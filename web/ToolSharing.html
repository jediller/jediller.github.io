CLI Tools for Bash and Video Slide Automation

Overview

This page links to CLI tools and relevant online communities for sharing workflows and improving scripts related to Bash, ffmpeg, and educational slide production with narration.

Communities to Share and Learn

r/ChatGPTPro
Power users who love sharing workflows
OpenAI Community Forum
The official forum — you can post projects, workflows, and tools there
r/commandline
Great subreddit for Unix/Linux shell users. Helpful crowd, lots of bash + ffmpeg fans
r/linux
Broader, but a strong place for utilities
r/bash
Niche but focused — good for your readySlides, writeVideo
Stack Overflow
If you want to post how-to or troubleshooting guides
GitHub
If you're up for maintaining a repo, even just a README and the scripts. Great for sharing and versioning

How to Use These Tools

  1. Prepare your slides as PNGs or JPGs named like slide1.png, slide2.png, etc.
  2. Prepare narration text as slide1.txt, slide2.txt, etc.
  3. Run readySlides to generate consistent slide images: readySlides --ann
  4. Run writeVideo to create the final MP4 lesson: writeVideo
