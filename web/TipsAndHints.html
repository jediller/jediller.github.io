Croping MP4 with ffmpeg


ffmpeg -i in.mp4 -filter:v "crop=out_w:out_h:x:y" out.mp4
Where the options are as follows:

out_w is the width of the output rectangle
out_h is the height of the output rectangle
x and y specify the top left corner of the output rectangle
see https://video.stackexchange.com/questions/4563/how-can-i-crop-a-video-with-ffmpeg






Turn video into slow motion



ffmpeg -i input.mkv -filter:v "setpts=2.0*PTS" output.mkv
=10.0  for ten times slower, =0.5 for twice as fast


see https://trac.ffmpeg.org/wiki/How%20to%20speed%20up%20/%20slow%20down%20a%20video

Creating Video from Image on Ubuntu


  Download images into directory
  Convert all the images to a smaller size. This step is only needed is the
  result is too large. Making the images smaller may reduce crispness.

  for f in `ls -1 IMG*.jpg`; do convert -resize 1248X702 $f 'G'$f; done

  ffmpeg -i IMG_20211205_112929068.jpg

  was used to get the size of the IMG, 1248X702 was choosen to keep the height
  and width ratios the same. This never hurts, and might be important in this
  case the original size was 4160X2340

  ffmpeg -framerate 1 -pattern_type glob -i 'GIMG*.jpg' -c:v libx265 -r 1 test1s.mp4

  creates a small video

  ffmpeg -framerate 1/3 -pattern_type glob -i 'GIMG*.jpg' -c:v libx265  -r 1 test1g.mp4

  creates a small video, but shows each image for 3 seconds

  even better

  ffmpeg -framerate 1/3 -pattern_type glob -i 'IMG*.jpg' -c:v libx265 -vf scale=1024:-1  -r 1 test7.mp4
  Without the -vf scale=1024:-1 the MP4 is 50MB with it only 4.5MB (dimensions of raw image 4160X2340)
